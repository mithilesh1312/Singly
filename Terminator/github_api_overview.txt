Markdown Cheat Sheet

Format Text

   Headers
# This is an <h1> tag
## This is an <h2> tag
###### This is an <h6> tag

   Text styles
*This text will be italic*
_This will also be italic_
**This text will be bold**
__This will also be bold__

*You **can** combine them*

Lists

   Unordered
* Item 1
* Item 2
  * Item 2a
  * Item 2b

   Ordered
1. Item 1
2. Item 2
3. Item 3
   * Item 3a
   * Item 3b
